# Pokedex React App

Welcome to the Pokedex React App! This app allows you to explore and search for Pokémon. It features a search bar to find specific Pokémon by name or ID, and a "Load More" button to fetch additional Pokémon data.

## Features

- View a list of Pokémon with basic information (name, ID, and image).
- Search for Pokémon by name or ID.
- Load more Pokémon to explore beyond the initial list.

## Getting Started

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/monurajputrko/Pokedex.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Pokedex
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

5. Open your browser and visit [http://localhost:3000](http://localhost:3000) to see the app in action.

## Usage

- Enter a Pokémon name or ID in the search bar to find specific Pokémon.
- Click the "Load More" button to fetch additional Pokémon data.
